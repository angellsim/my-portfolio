## Portfolio - Alice Wolf Fernandes

Portfolio em Next.js com tema inspirado no personagem Satoru Gojo de Jujutsu Kaisen.

## Getting Started

Para acessar o portfólio, basta clicar [AQUI](https://angellsim.github.io/my-portfolio/)

Caso tenha problemas em acessar o link, tente os seguintes passos

1. Rode o servidor local:

```bash
npm install
npm run dev
```

2. Abra [http://localhost:3000](http://localhost:3000).
